2048 Game with Wallet Connection
1. Unzip the folder.
2. Open index.html in a browser (Chrome, Firefox, etc.).
3. Use arrow keys to play 2048.
4. Tap "Connect Wallet" for MetaMask (install MetaMask app first).
5. Tap "Reset Game" to start over.
6. Check browser Console (developer tools) for errors.
Note: Keep all files in the same folder.
